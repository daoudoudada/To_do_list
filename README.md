# To Do List

Aplicación web sencilla para gestionar tareas (to-do) hecha con **HTML, CSS y JavaScript**. Permite agregar, editar, marcar como completadas y eliminar tareas. Los datos se guardan en el navegador usando **localStorage**.

## Funciones
- Crear tareas con **texto**, **prioridad** y **hora**
- Editar tareas
- Marcar como completadas / pendientes
- Eliminar tareas
- Filtro: todas / pendientes / completadas
- Contador de tareas pendientes
- Guardado automático en localStorage

## Tecnologías
- HTML
- CSS
- JavaScript (vanilla)

## Cómo ejecutar
1. Descarga o clona el proyecto.
2. Abre `index.html` en tu navegador.

Opcional (servidor local):
```bash
python -m http.server 8000
```
Luego entra a `http://localhost:8000`.
